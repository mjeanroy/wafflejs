## `toString()`

### Usage

Joins the array and returns one string containing each array element separated by commas.

```javascript
// Get index of object.
var val1 = collection.toString();
```

### Arguments

None.

### Returns

| Type       | Description                          |
|------------|--------------------------------------|
| `string`   | Elements joined as a single `string` |

### Running Time

`O(n)`
