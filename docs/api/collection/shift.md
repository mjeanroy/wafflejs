## `shift()`

### Usage

Removes the first element from the collection and returns that element.
If you call `shift()` on an empty collection, it returns an `undefined` value.

```
var first = collection.shift();
```

### Arguments

None.

### Returns

| Type       | Description      |
|------------|------------------|
| `object`   | Removed element. |

### Running Time

`O(n)`
